### 1. **StudentProfile** (User extension)

- full_name, department, year, phone, address
- guardian_name, guardian_phone
- timestamps (created_at, updated_at)
- Properties: has_allocated_room, allocated_room

### 2. **Room** (Hostel management)

- room_number (unique), block_name, floor
- capacity, current_occupancy
- room_type (Single/Double/Shared), status
- amenities, timestamps
- Methods: update_occupancy(), Properties: available_slots, occupancy_percentage, is_full

### 3. **RoomAllocation** (Application workflow)

- student (FK), room (FK), status
- applied_date, allocated_date, rejection_reason
- Methods: approve(), reject()
- Constraints: unique_together('student', 'room')

### 4. **Complaint** (Issue tracking)

- student (FK), subject, description
- status, priority, room (FK optional)
- resolution_notes, timestamps
- Method: resolve()
